# Tricks to move around quickly in R


[Tips for working in the R Console](https://support.rstudio.com/hc/en-us/articles/200404846-Working-in-the-Console)

```
CTRL + Up  #does the trick
```

```
CTRL + L  # clear the console
```

```
**Ctrl+1** — Move focus to the Source Editor
**Ctrl+2** — Move focus to the Console
**Ctrl+L** — Clear the Console
**Esc** — Interrupt R
```
